### Github blog 创建

  免费的个人blog网站。

### hexo 搭建

  用于生成静态网页。

### hexo 关联 github

  通过hexo命令自动将网页数据发布到GitHub的个人blog上。

### 备份原始数据

  hexo 生成的配置文件和自己编写的数据文件需要通过原始git仓库保存。

```shell
git checkout -b hexo
git push origin hexo
```
