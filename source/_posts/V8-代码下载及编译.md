## 背景

  为了学习和了解 Javascript Engine 的工作原理，在本地下载了V8的代码。

 <!-- more -->

## 环境

1. 硬件平台：x86_64
2. OS: ubuntu 20.04

## 下载编译

1. 安装depot_tools

```
git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git
```

2. 在空目录下拉取 V8 代码 

```
fetch v8
```

3. 更新工具包

```
gclient sync
```

4. 在 V8 的代码根目录下更新编译所需工具

```
./build/install-build-deps.sh
```

5. 编译

```
tools/dev/gm.py x64.release
```
